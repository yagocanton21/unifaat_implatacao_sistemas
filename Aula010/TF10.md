# TF10 - Migração de Banco Local para RDS

**Disciplina:** Implementação de Sistemas  
**Curso:** Análise e Desenvolvimento de Sistemas - UniFAAT  
**Valor:** 2,0 pontos  
**Tipo:** Trabalho prático individual  
**Prazo:** Conforme cronograma da disciplina

## Objetivo

Demonstrar competência na migração de um banco de dados local para Amazon RDS, implementando alta disponibilidade, backup automatizado e monitoramento, comparando performance e custos entre as duas abordagens.

## Descrição da Atividade

Você deve migrar o Banco de dados (docker-compose.yml) deste repo para utilizar Amazon RDS, documentando todo o processo, comparando performance e implementando boas práticas de segurança e monitoramento.

## Cenário do Projeto

### Arquitetura Atual vs Nova
```
ANTES (Local): Postgres

DEPOIS (RDS):
Amazon RDS Postgres (Multi-AZ)
```
